# BMI Calculator

## Description
The application is a calculator which can give you your BMI in both Metric and Imperial method.

## User Story
```
As a User,       
In order to know if i'm healthy,     
I want to be able to get my BMI.
```

## Dependencies
- React
- Cypress
- Enzyme
